### 饼图外部标签  
做的APP里插入饼图后，外部的标签文字有些不能完全显示在屏幕中，看了官方文档没找到属性后差点放弃想要把标签放在图内，突然发现官方例子中有写到，真是开心。

在series里面加上这个属性，控制标签的长度就好啦  

	itemStyle : {
                        normal : {
                            label : {
                                show : true
                            },
                            labelLine : {
                                show : true,
                                length:5
                            }
                        }
                        
                   }
